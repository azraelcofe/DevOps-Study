1、使用Git Bash生成证书

ssh-keygen.exe -t rsa

$cd ~ //进入用户家目录
$ls
id_rsa id_rsa.pub

查看文件内容
$ cat id_rsa.pub

2.把公钥id_rsa.pub的内容放置到github上。

打开Github。点击用户图像--Settings--SSH and GPG keys--New SSH

3.克隆仓库代码

git clone git@github.com:azraelcofe/DevOps-Study.git

4.编写代码

cd ~/Desktop/devops
git status
git add README.md
git commit -m "第一次提交" README.md
git push
